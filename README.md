## Ejercicios

### 1. Sintaxis Básica y Variables
- Mensaje de bienvenida con nombre del grupo
- Declarar e imprimir variables: `int`, `string`, `double`, `bool`

### 2. Estructuras de Control
- Verificar si un número es par o impar
- Bucle `for` para imprimir números del 1 al 10

### 3. Clases y Objetos
- Clase `Persona` con propiedades `Nombre` y `Edad`
- Método `Presentarse()`
- Crear 3 objetos (uno por integrante)

### 4. Herencia
- Clase base `Animal` con método `HacerSonido()`
- Clases hijas: `Perro` (guau) y `Gato` (miau)

### 5. Polimorfismo
- `List<Animal>` con diferentes tipos de animales
- Recorrer lista llamando `HacerSonido()`

### 6. Encapsulamiento
- Clase `CuentaBancaria` con atributo privado `saldo`
- Métodos `Depositar()` y `ObtenerSaldo()`

### 7. Interfaces y Abstracción
- Interfaz `IVehiculo` con métodos `Encender()` y `Apagar()`
- Implementar en clases `Carro` y `Motocicleta`
